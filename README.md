# Privacy Policy for To Do List Web Application

## Project Description

This is a simple web application that displays the privacy policy for a To Do List app. The page is built using HTML and CSS, providing a clean and professional layout for users to understand how their data is handled.

**Live Demo:** [https://itsiamdev.github.io/Politica-de-confidentialitate/](https://itsiamdev.github.io/Politica-de-confidentialitate/)

## Features

- Responsive design that works on desktop and mobile devices
- Clean, modern UI with a professional appearance
- Comprehensive privacy policy covering data collection, usage, storage, and user rights
- Contact information for privacy-related inquiries

## Technologies Used

- HTML5
- CSS3
- JAVA SCRIPT

## Installation

1. Clone or download the repository
2. Open `index.html` in any modern web browser

No additional dependencies or server setup required.

## Usage

Simply open the `index.html` file in your web browser to view the privacy policy page.

## Structure

- `index.html` - Main HTML file containing the privacy policy content
- `style.css` - CSS file for styling the page
- `README.md` - This file with project documentation
